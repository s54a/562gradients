Gradients
This WebPage has 562 Gradients which have been taken from these website Web Gradients & UI Gradients.

Press Next & Previous Buttons to Navigate.

You can enter a number in the Input Box which is beside the set Button to move in the List.

The Colors of the Random Gradient Colors are logged in Console and they are not saved else where.

Copy Button will copy the Object which has the Information about the Gradient to the clipboard.
